# iOS-System-Symbols-Supplement
This repository is a supplement to [Zuikyo/iOS-System-Symbols](https://github.com/Zuikyo/iOS-System-Symbols).

You can get versions prior to 12.0 here [Zuikyo/iOS-System-Symbols/collected-symbol-files](https://github.com/Zuikyo/iOS-System-Symbols/blob/master/collected-symbol-files.md), and this repository will update versions later than 12.0.

Thanks to [Zuikyo](https://github.com/Zuikyo).
